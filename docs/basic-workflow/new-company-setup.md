# 新公司配置

新公司创建完成后，建议先完成以下配置，确保后续业务流程顺畅：

- 员工权限配置：创建角色并分配权限，保障不同岗位的操作边界。
- 品牌车型配置：维护公司自有或代理的品牌、车型字典，方便车辆入库。
