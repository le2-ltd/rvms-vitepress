# 续租

续租是在当前合同未结算前，继续使用同一车辆并延长服务周期；原合同保持不变，新合同视为服务的延伸，可调整价格、周期等条款，并可追加押金与管理费。

## 适用范围

- 短期、长期合同均可在未结算前申请续租。
- 同一合同可多次续租，只要服务未结束。

## 操作流程

1. 申请续租：在原合同未结算状态下发起续租。
2. 创建续租合同：，新合同司机、车辆保持不变，可调整租期、租金、管理费等条款，同时支持增加押金。
3. 收款与确认：在新合同签约收款时，补充押金/管理费并确认生效。
4. 连续服务：多个续租周期之间不单独结算，保持服务连续。
5. 统一结算：最后一个续租周期结束后，再统一做结算并退回押金。

## 注意事项

- 续租不改变原合同记录，仅在其基础上新增续租合同。
- 押金可在续租时追加，在最终结算时一并处理。
