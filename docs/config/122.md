# 122 账号设置

通过设置「智行租赁」的 122 账号信息，可以导入车辆违章信息。同时还可以导入车辆列表。

前提：只能查询本公司的车辆违章信息。

## 如何获取 cookie

登录交管网 122 账号，获取登录状态 cookie：

（1）登录网站，选择单位登录，在交管 12123APP 上扫码登录单位账号（如果是多个单位账号，chrome 浏览器一定要用无痕模式/inPrivate 打开窗口）
![](assets/1.png)
![](assets/2.png)
![](assets/3.png)

（2）登录成功后，点击鼠标右键，点击【检查】或按 F12，在右侧开发页面点击【网络】，然后点击【Fetch/XHR】
![](assets/4.png)

（3）按照下图的操作步骤，复制 Cookie 中的全部内容
![](assets/5.png)

（4）复制出来后，粘贴到 cookie 输入框内即可。交管网的网页直接关闭，千万不用点网页上的【退出】，要不然又得重新操作了。
![](assets/6.png)

## 新增 122 账号后

1. 系统自动对 cookie 展期，无需人工操作。
2. 系统每日自动查询违章查询，并记录到系统里。
3. 系统通过违章时间与「发车验车、退车验车」的时间，把每个违章关联到租赁订单上，也就能对应出准确的司机。
