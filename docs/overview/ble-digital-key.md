# 智行租赁蓝牙数字钥匙产品说明书

## 1. 产品概述

智行租赁蓝牙数字钥匙是一款面向汽车租赁公司的车载智能设备，集成 GPS 定位监控、4G 通信、蓝牙数字钥匙和订单自动授权能力。

设备安装后，租车公司可在系统中查看车辆位置、接收越界报警，并根据订单合同、支付款项、退车状态等业务信息，自动向租客授权车辆解锁、上锁、寻车/鸣笛、后备箱等车辆钥匙能力。

产品适用于：

1. 短租、长租、以租代购等租赁车辆管理场景。
2. 需要减少实体钥匙交接、降低门店值守压力的租车公司。
3. 需要结合订单状态自动控制车辆使用权限的运营团队。
4. 需要对车辆越界、异地用车、逾期未还等风险进行提醒的风控场景。

## 2. 核心能力

### 2.1 GPS 定位监控

设备支持 GPS 定位监控能力，可用于车辆位置查看、车辆轨迹管理和运营风控。

定位能力说明：

1. 设备支持多星座卫星定位，日常可稳定搜索约 30 颗卫星。
2. 定位结果具备较高精度，可满足租赁车辆日常监管、找车、风控提醒等场景。
3. 实际定位精度会受到车辆停放环境、建筑遮挡、地下车库、天气、信号质量等因素影响。
4. 对于室内、地下、隧道、高楼密集区域等弱信号环境，定位可能出现延迟或偏移，属于卫星定位类设备的常见现象。

### 2.2 4G 在线通信

设备通过 4G 网络与智行租赁平台保持通信，用于上传车辆定位、接收授权状态、同步设备状态和触发风控通知。

租车公司无需依赖车辆附近的 Wi-Fi 环境，只要设备所在区域具备可用 4G 信号，即可与平台保持连接。

### 2.3 蓝牙数字钥匙

租客可通过微信小程序使用蓝牙数字钥匙能力，对车辆执行常用钥匙操作。

当前支持：

1. 解锁车辆。
2. 上锁车辆。
3. 寻车/鸣笛。
4. 打开后备箱。

在此基础上，只要原车遥控钥匙本身具备对应按键功能，蓝牙数字钥匙即可通过改造方案实现相应控制。

蓝牙数字钥匙适合车辆附近操作场景。租客到达车辆附近后，通过微信小程序连接设备并发送钥匙指令，设备模拟原车遥控钥匙按键完成对应动作。

支持弱网/无网用车场景。租客在授权有效期内，即使手机临时没有网络，也可以通过微信小程序读取本地有效钥匙凭证，并在车辆附近使用蓝牙完成开关车操作。

说明：不同车型的原车钥匙功能不同，最终可支持的车辆动作以原车遥控钥匙实际按键能力和改造结果为准。

## 3. 自动电子围栏

系统内置全国主要城市边界数据，每台设备可绑定一个运营城市。

当车辆越过绑定城市边界时，系统会自动触发电子围栏报警，并由企业微信机器人通知到指定企业微信群。该群可由车管、驾管、销售、客户相关人员组成，便于车辆风险被及时发现和协同处理。

电子围栏适合以下运营场景：

1. 限制车辆跨城市使用。
2. 发现租客异常行驶范围。
3. 辅助租车公司处理异地用车、逾期未还、风险订单等问题。

说明：

1. 电子围栏基于系统内置城市边界和设备定位数据判断。
2. 报警触发效果受定位信号、边界数据精度和车辆实时通信状态影响。
3. 如车辆处于弱信号区域，报警可能存在延迟。

## 4. 租客用车与自动授权

### 4.1 订单自动授权

系统会根据订单合同、支付款项、退车等信息自动控制租客的数字钥匙权限。

典型规则：

1. 订单满足用车条件后，系统自动向租客授权数字钥匙。
2. 租客可在微信小程序中对车辆执行解锁、上锁、寻车/鸣笛、后备箱等操作。
3. 订单结束、退车完成或不再满足用车条件后，系统自动取消授权。
4. 系统使用人员无需再手动开通权限，也无需在退车后手动取消权限。

该授权能力仅面向租客使用。公司员工、管理员、维修人员等其他角色的用车权限应通过单独机制处理，避免不同角色权限混用，降低安全风险。

### 4.2 微信小程序使用

租客通过微信小程序使用车辆数字钥匙，无需额外安装 App。

## 5. 安装方式

### 5.1 设备安装

设备通过车辆 OBD-II 接口取电，安装过程不需要破坏原车线束。OBD-II 也常被称为 OBD2，是车辆常见的诊断接口。

安装优势：

1. 不破线，不改原车供电线路。
2. 安装位置明确，便于标准化施工。
3. 后期维护、检测和更换更方便。
4. 相比需要破线接电的改装方案，对车辆原线路影响更低。

安装分工：

1. OBD-II 设备安装由租车公司自行完成。
2. 智行租赁提供设备安装指导。
3. 因设备通过 OBD-II 接口取电，安装步骤简单，适合租车公司批量自行安装。

### 5.2 蓝牙数字钥匙集成

蓝牙数字钥匙需要对原车遥控钥匙进行专业集成改造。

改造完成后，设备通过控制电路模拟原车遥控钥匙按键，实现解锁、上锁、寻车/鸣笛、后备箱等动作。车辆端取电仍通过 OBD-II 接口完成，不需要破坏原车线束。

服务方式：

1. 智行租赁可提供焊钥匙安装服务。
2. 租车公司也可自行安排当地师傅焊接钥匙。
3. 如客户自行安排师傅，智行租赁提供必要的安装指导。

注意事项：

1. 焊接钥匙应由专业人员操作。
2. 不同车型、不同钥匙结构可能存在施工差异。
3. 租车公司应提前确认车辆是否具备可改造的原车遥控钥匙。
4. 焊接完成后应现场测试解锁、上锁、寻车/鸣笛、后备箱等按键动作是否稳定。

## 6. 安全机制

蓝牙数字钥匙涉及车辆控制，智行租赁在通信和授权层面做了安全设计。

### 6.1 订单授权控制

数字钥匙权限由系统根据订单状态自动发放和回收。

安全价值：

1. 租客只能在符合订单条件时使用车辆。
2. 退车完成或订单失效后，租客权限自动取消。
3. 避免系统使用人员忘记取消授权造成后续用车风险。
4. 不与公司员工、管理员、维修人员等其他角色共用授权逻辑。

### 6.2 一机一密

设备与小程序之间采用一机一密机制，不同设备使用不同通信密钥。

安全价值：

1. 降低单一密钥泄露影响范围。
2. 避免不同车辆设备之间复用同一通信凭证。
3. 提升设备级别的访问隔离能力。

### 6.3 加密通信

蓝牙钥匙指令采用加密通信机制，避免明文发送车辆控制指令。

通信安全设计包括：

1. AES-128 加密。
2. 每次钥匙操作前获取随机数。
3. 随机数一次一用，连续操作需要重新获取随机数。
4. 命令帧校验，异常数据可直接丢弃。
5. 指令串行执行，避免并发命令造成状态混乱。

### 6.4 操作确认

小程序发送钥匙指令后，设备会返回对应命令结果。小程序侧可根据设备响应判断操作是否完成，减少重复点击、误判和状态不一致问题。

## 7. 售后与质保

### 7.1 设备质保

1. 设备硬件按常规电子设备提供 1 年质保服务，具体以合同或报价单约定为准。
2. 质保期内，如设备出现非人为硬件质量问题，经检测确认后可提供维修或换新处理。
3. 因人为损坏、进水、私自拆改、异常电压、车辆线路异常、外力撞击等原因造成的损坏，不属于免费质保范围。
4. 到货后应及时核对设备数量、外观和通电状态；如存在运输损坏或到货异常，应及时反馈处理。

### 7.2 钥匙改造与安装支持

1. 智行租赁可提供焊钥匙安装服务，具体费用按实际服务方案确认。
2. 租车公司自行安排师傅焊接钥匙时，智行租赁提供必要指导，但施工质量由实际施工方负责。
3. 钥匙焊接完成后，应在现场完成所有按键功能测试，确认后再投入运营。
4. 如车辆更换、钥匙更换或车型差异导致需要重新改造，相关费用按实际情况另行确认。

### 7.3 平台与流量服务

1. 服务期内，设备可正常接入智行租赁平台并使用定位、电子围栏、订单授权等能力。
2. 平台服务和设备流量期限以合同或报价单约定为准。
3. 服务到期后的续费方式以届时实际方案为准。

## 8. 使用注意事项

1. 安装前确认车辆 OBD-II 接口位置、原车遥控钥匙状态和车型适配情况。
2. 安装完成后，必须测试定位上线、蓝牙连接、解锁、上锁、寻车/鸣笛、后备箱和企业微信通知能力。
3. 每台设备应绑定正确车辆和运营城市，避免电子围栏误报或漏报。
4. 订单、付款、退车信息应及时录入系统，确保数字钥匙自动授权准确生效。
5. 租客反馈无法开关车时，系统使用人员应优先检查订单权限、车辆附近蓝牙连接、设备在线状态和车辆电瓶状态。
6. 车辆进入地下车库、隧道或信号弱区域时，定位和报警可能出现延迟，应结合业务情况判断。
