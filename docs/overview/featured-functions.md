# 智行租赁特色功能

## 查询违章、年检

1. 支持 122 账号状态管理。
2. 每日自动批量获取公司车辆，查询本公司车辆列表、车辆违章、年检。
3. 违章信息可自动推送到租客端。

## 合同模板

1. 支持租车合同、结算单、财务收据、验车单等 DOCX 文件模板。
2. 可以根据不同的租赁模式定制不同的合同模板。
3. 生成 DOCX、PDF 格式的合同文件。

## 证件自动识别

1. 支持行驶证、驾驶证、身份证等证件的图片识别。
2. 支持部分交强险 PDF、商业险 PDF 文件的内容提取和字段归纳。

## 菜单权限控制

1. 客户端每个菜单、按钮都对应权限，只有拥有相应权限的登录者才能看到。
2. 服务器每个请求都对应权限控制，安全性做到最高。

## 不限容量图片记录

1. 验车、维修、保养、出险等功能上传图片不限制数量。
2. 系统自动清理未使用的图片。

## 订单全链条关联

1. 每个违章、维修、保养、出险、财务记录都可能会关联到一个租赁订单上。
2. 系统通过换车、进出验车记录的时间自动计算行程，违章通过行程关联，确保违章信息准确无误。
3. 同时，还支持车辆维度和客户维度的多维度关联分析。
