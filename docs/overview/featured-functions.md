# 智行租赁特色功能

## 订单全链条关联

1. 违章、维修、保养、出险、财务流水等记录全部与租赁订单自动绑定，形成全生命周期档案。
2. 依据换车、验车时间自动生成行程，将违章精准定位到责任车辆与司机。
3. 提供车辆、客户、订单等多维度穿透视图，为运营优化与风险预警提供决策依据。

## 查询违章、年检

1. 深度对接「交管 122」账号，实时监测账号状态与 Cookie 失效，保障同步链路稳定。
2. 每日定时批量拉取车辆列表、违章记录与年检结果，彻底告别人工登录导出。
3. 违章信息自动匹配司机并推送至租客端，提醒、处置、留痕形成闭环。

## 合同模板

1. 预置租车合同、结算单、财务收据、验车单等 DOCX 模板，开箱即用还可复制扩展。
2. 可按长租、短租、以租代购等模式自由组合条款字段，兼顾灵活度与合规性。
3. 支持一键导出 DOCX/PDF，线上审批与纸质归档同时提速。

## 证件自动识别

1. 对行驶证、驾驶证、身份证等证件图片进行 OCR，自动抽取关键信息并校验有效期。
2. 支持交强险、商业险 PDF 的内容解析与字段归纳，识别结果直接写入系统，减少手动录入。

## 菜单权限控制

1. 客户端菜单、按钮与服务端接口逐一绑定权限粒度，账号只会看到获批操作。
2. 请求层面再做一次权限校验，防止越权访问，轻松满足审计与安全要求。

## 不限容量图片记录

1. 验车、维修、保养、出险等流程支持不限量图片上传，完整保留可追溯证据链。
2. 系统自动清理未引用图片文件，既保证留痕又控制存储成本。
