# 智行租赁特色功能

## 查询违章、年检

1. 深度接入「交管 122」账号体系，监控账号状态、Cookie 失效等异常。
2. 每日定时批量获取公司车辆列表、违章记录与年检状态，减少人工查询。
3. 违章结果可一键推送至租客端，客户及时知悉、闭环处理。

## 合同模板

1. 支持租车合同、结算单、财务收据、验车单等 DOCX 模板。
2. 根据长租、短租业务模式，自由组合条款与字段。
3. 一键导出 DOCX、PDF，线上审批、打印存档都更高效。

## 证件自动识别

1. 识别行驶证、驾驶证、身份证等多类证件图片，智能抽取关键信息。
2. 支持部分交强险、商业险 PDF 的内容解析与字段归纳，自动落库。

## 菜单权限控制

1. 客户端的菜单、按钮与服务端接口均绑定权限粒度，账号只能看到授权内容。
2. 请求层面二次校验，防止越权访问，数据安全与合规同步提升。

## 不限容量图片记录

1. 验车、维修、保养、出险等流程支持不限数量的图片上传，完整保留证据链。
2. 定期清理未引用的图片文件，在保证留痕的同时让存储可控。

## 订单全链条关联

1. 违章、维修、保养、出险、财务等记录都可绑定到对应租赁订单，形成完整档案。
2. 系统依据换车、验车时间自动生成行程，将违章精准定位到车辆和驾驶人。
3. 支持以车辆、客户、订单等多维度穿透分析，为运营决策提供依据。
