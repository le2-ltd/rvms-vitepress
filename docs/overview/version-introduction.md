# 版本介绍

智行租赁提供轻量、标准、旗舰三档方案，功能模块全部放开，均包含管理端网页、管理端小程序以及租客端微信小程序，企业可按车队规模灵活选择。

- 轻量版：适合 50 台车以内团队，¥1,300/年，覆盖基础运营所需能力。
- 标准版：适合 400 台车以内团队，¥5,200/年，满足多组织协同与精细管理。
- 旗舰版：面向 400 台车以上企业，¥10/台车/年，支持更高并发与复杂流程。

所有方案均采用独立部署，提供专属二级域名，并默认开启数据与文件自动备份，确保业务安全稳定运行。
