# 智行租赁有哪些功能

## 员工管理

### 登录方式

- 支持账号 + 密码登录，可配置验证码、防暴力破解等安全策略。

### 角色管理

- 系统内置系统管理员、经理、车管、销售、财务、修理厂等典型角色，开箱即用。
- 角色权限支持自定义增删。

### 管理员管理

- 管理员可被授予多个角色；权限组合实时生效，便于实现职能交叉。
- 内置超级用户默认拥有全部权限，用于初始配置与紧急排障。

## 数据统计

### 数据统计

- 支持日、周、月、年等多维度统计，不同角色仅查看有权限的数据。
- 指标覆盖租赁订单数量与金额、结算次数、计划/实际收付款次数与金额、验车次数、维修数量与金额等核心指标，帮助实时掌握运营健康度。

## 车辆管理

### 车型管理

- 支持车型信息录入，状态可切换。

### 车辆管理

- 支持车辆基础信息、证件档案、附件资料的集中录入。
- 通过行驶证 OCR 自动识别字段，降低手工输入错误。
- 车辆上架、签约、验车、换车、退车等轨迹按时间线沉淀，随时追溯。

## 客户管理

### 客户信息管理

1. 支持个人与企业客户两种类型，记录联系人、紧急联系人、身份证/驾驶证、人证、企业资质等信息。
2. 上传身份证、驾驶证图片即可自动识别字段。
3. 自动筛出证件即将到期的客户，便于提前提醒。

### 司机租车全记录

1. 基于客户维度汇总订单、收付款、退车、验车、行程、维修、违章等历史数据，帮助快速了解风险画像。

## 租车、退车流程管理

1. 租车：（销售）新建待签约 → (财务)确认收款 → (车管)发车验车。
2. 退车：(车管)退车验车 → (车管)新建退车结算 → (财务)审核结算 → (财务)退押金。
3. 系统严格校验前置步骤，确保流程合规可控。
4. 预订管理:未到期车辆进入预定车辆清单里后，可以方便顾客提早了解车况,租金等信息，缴纳定金即可提早锁定车辆。
5. 车辆签约前，需要走完「车辆整备」流程。

### 订单操作、状态

1. 初始状态为「待签约」
2. 签约收款后订单状态为「已签约」
3. 作废操作后订单状态为「已作废」
4. 结算后订单状态改为「提前解约」或「已完成」
5. 在签约前，可以换车。

### 维修/保养/年检/保险/事故管理

1. 为维修、保养、出险、年检、保险保单、手动违章提供统一台账与图片留痕。
2. 可为合作修理厂开通账号，实时掌握外部维修进度。
3. 亦可为合作保险公司开启账号，用于事故处理、报价与结算协同。

## 财务管理

### 收付款账号管理

- 可管理多组收付款账号，系统自动核算余额，方便对账。

### 收付款类型管理

- 内置上百种收付款类型，可按需启用。

### 收、退押金与收租金

- 首期租金、押金、管理费收款完成后合同自动生效。
- 支持逐期记录租金实收情况，并与计划表联动。

### 其他类型收付款

- 适用于违章、维修、赔付等其他流水，形成完整财务档案。

## 风控管理

### 风险一览表

- 依据年检、保险、频繁违章、证照到期、租金欠付等数据，自动筛选风险车辆与客户，帮助提前预警。

## 消息管理

1. 系统深度绑定企业微信。发送消息的渠道包括：消息推送（群机器人）、应用推送消息.
2. 内置消息类型：车辆违章处理提醒、年检提醒、车辆续保提醒、租客合同到期提醒、租客支付租金提醒，可以配置发送内容模板，提早天数、状态等。
3. 集中配置企业微信群机器人、企业微信用户。
4. 查看所有发送的消息，包括发送时间和状态。

## 租客端微信小程序

### 车辆管理

- 查看手动录入的违章、平台同步的违章，以及维修、保养、出险记录。

### 销售管理

- 随时查询租赁订单进度与状态。

### 财务管理

- 查看应收/实收、押金等收付款记录，按计划按期缴费。

### 风控管理

- 提醒待年检车辆、司机证照到期、车证到期等风险事项。

### 订单合同管理

- 支持创建长租、短租等多种组合合同，支持作废、提前解除等操作。

### 付款计划管理

- 长租合同可根据周期、期限、租金、押金、管理费自动生成收款计划表，并同步财务应收、实收记录。
